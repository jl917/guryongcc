# Utils

## Partial (部分 부분적)

```ts
interface Todo {
  title: string;
  description: string;
}

type Todo2 = Partial<Todo>

type Todo2 = {
  title?: string | undefined;
  description?: string | undefined;
}
```

## Required

```ts
interface Todo {
  title?: string;
  description?: string;
}

type Todo2 = Required<Todo>

type Todo2 = {
  title: string;
  description: string;
}
```

## Readonly

```ts
interface Todo {
  title: string;
}

type Todo2 = Readonly<Todo>

type Todo2 = {
  readonly title: string;
}
```
