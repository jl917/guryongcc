## 재귀
##### 콜 스택 제한
스택 오버플로우를 자바스크립트로 처리할 수 있지만 권장하지 않음. 콜 스택 크기를 초과할 가능성이 있는 코드는 배포하면 안됨.
##### 재귀 패턴
```javascript
function recurse(){
    recurse()
}
recurse()
```
##### 반복
재귀로 구현할 수 있는 알고리즘은 모두 반복으로 구현 가능, 재귀함수는 종종 스택오버플로우 에러를 일으킨다.
