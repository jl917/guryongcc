## 条件判断语句
##### if...else
```javascript
if(x == 0){
    //执行语句
}else if(x == 1){
    //执行语句
}else{
    //执行语句
}
```

`false`值等效

- false
- undefined
- null
- 0
- NaN
- ""

##### switch
```javascript
switch(expression){
    case: "label_1":
        //执行语句
        break;
    case: "label_2":
        //执行语句
        break;
    default:
        //执行语句
        break;
}
```
